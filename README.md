# Задание

## Команда (класс) должна получать на вход в качестве аргумента хэш, например, следующего вида:

```
{
  a: :a1,
  b: [:b1, :b2],
  c: {c1: :c2},
  d: [:d1, [:d2, :d3], {d4: :d5}],
  e: {e1: {e2: :e3}, e4: [:e5, :e6]}
}
```

## И на выходе отдавать хэш вида:

```
{
a: {a1: {}},
b: {b1: {}, b2: {}},
c: {c1: {c2: {}}},
d: {d1: {}, d2: {}, d3: {}, d4: {d5: {}}},
e: {e1: {e2: {e3: {}}}, e4: {e5: {}, e6: {}}}
}
```

### Вложенность хэша может быть любой (не ограничена).
